teamform

A real-time online team forming app for any event (e.g. COMP3111, hackUST, ...)



Enter a name to create/join an event


The base code is NOT perfect. It is mainly used to illustrate how to do programming in AngluarJS/Firebase. If you find bugs in the base code, please open an issue at Github. If you don't like the base code, feel free to rebuild everything from scartch.

The demo application can also be used to form teams in COMP3111/H. Please enter COMP3111 or COMP3111H as an event name. For the user ID, please enter your ITSC email. For example, if your ITSC email is cspeter@connect.ust.hk, please enter cspeter

What is teamform?
A real-time online team forming app

The following roles are supported in teamform:

Administrator

  • Be responsible for creating an event for team forming (e.g. COMP3111, COMP3111H, hackUST)
  • Be able to set the minimum and maximum team size for the event (e.g. 5..7 team members in COMP3111 and 5..8 team members in COMP3111H)

Team

  • Be able to add and remove requests made by members
  • Once a member joins the team, all his/her requests will be deleted

Member

  • Be able to request to join multiple teams
